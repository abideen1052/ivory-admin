# Notification Admin Dashboard

A simple React web admin panel to manage notifications.

## Features

- **List View**: Table view of notifications.
- **Actions**: A "Resend" button (mocked).
- **Create**: A simple form to add a new notification (mocked).

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS v4

## How to Run

```bash
npm install
npm run dev
```
